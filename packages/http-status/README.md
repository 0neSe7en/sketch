# http-status

Provides most useful http status.

```sh
  ExtendableError,
  Forbindden,
  Gone,
  InternalError,
  InvalidRequest,
  NotAcceptable,
  NotFound,
  Unauthorized,
  UnprocessableEntity,
```

## Install

```bash
yarn add http-status
```

## Usage

```js
import { Forbindden } from "@36node/http-status";

throw new Forbidden();
```
